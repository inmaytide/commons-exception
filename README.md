## commons-exception
- Global exception handler for "Spring MVC&Spring WebFlux"

## Getting started

-   Add MyBatis-Plus dependency
    -   Maven:
        ```xml
        <dependency>
            <groupId>com.baomidou</groupId>
            <artifactId>mybatis-plus-boot-starter</artifactId>
            <version>3.2.0</version>
        </dependency>
        ```
